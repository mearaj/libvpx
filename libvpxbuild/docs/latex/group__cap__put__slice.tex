\doxysubsubsection{Detailed Description}
The following function is required to be implemented for all decoders that advertise the VPX\+\_\+\+CODEC\+\_\+\+CAP\+\_\+\+PUT\+\_\+\+SLICE capability. Calling this function for codecs that don\textquotesingle{}t advertise this capability will result in an error code being returned, usually VPX\+\_\+\+CODEC\+\_\+\+INCAPABLE.

\doxysubsubsection{Detailed Description}
vp9 svc frame flag parameters. 

This defines the frame flags and buffer indices for each spatial layer for svc encoding. This is used with the \mbox{\hyperlink{group__vp8__encoder_gga6deae3d561c838952552c3d3756322ecaa0d822d0d1efa1868ce05d8523a8836b}{VP9\+E\+\_\+\+SET\+\_\+\+SVC\+\_\+\+REF\+\_\+\+FRAME\+\_\+\+CONFIG}} control to set frame flags and buffer indices for each spatial layer for the current (super)frame.
